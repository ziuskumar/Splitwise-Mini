# Shared Expenses App

A full-stack "Shared Expenses" application (like a mini Splitwise) designed to manage shared expenses for groups whose membership changes over time.

## Architecture Overview

- **Backend**: Django 5 + Django REST Framework + SimpleJWT Authentication + PostgreSQL
- **Frontend**: React (Vite) + TailwindCSS + React Router + Axios + Recharts
- **Database**: PostgreSQL (with dynamic membership queries and transaction-per-row CSV importing)
